Short Bio

Projects

I have been worked on several exciting projects related to Big Data Processing Systems and Cloud Computing:

Mammoth

An autonomic data processing framework for scientific state-transition applications. Mammoth provides a MapReduce-like programming model for state-transition applications and compiles users’ code into highly-optimized Hadoop jobs.

Palantir

An Software-Defined Network (SDN) service for big data frameworks, which automatically and dynamically maintain locality information.

Gandalf

This on-going project focuses on improving multi-stage data-parallel DAG scheduler for Spark.

GatorMap

This project enhances MapReduce system by optimizing the incremental processing of iterative MapReduce jobs and leveraging cluster affinity to optimize the task scheduling
