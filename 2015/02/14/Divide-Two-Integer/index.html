Divide Two Integer

Problem

Divide two integers without using multiplication, division and mod operator.
If it is overflow, return MAX_INT.

Solution

Corner cases are: Integer.MIN_VALUE / -1, div by zero, negative.
I use long to avoid overflow.

About the math, easy: $\frac{x}{y} = \frac{x \gg 1}{y} + \frac{x \gg 1}{y} + \frac{x - x \gg 1 - x \gg 1}{y}$. So use binary search to find $\frac{x \gg 1}{y}$ and then resemble the result.
