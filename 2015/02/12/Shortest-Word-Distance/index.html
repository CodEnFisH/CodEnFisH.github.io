Shortest Word Distance

Problem

Please design and implement a class:
This class will be given a list of words (such as might be tokenized from a paragraph of text), and will provide a method that takes two words and returns the shortest distance (in words) between those two words in the provided text.

Solution

Say the two words are s1 and s2, we just need two pointers to record the position where we see s1 and s2. The answer is to minimize the gap between the two pointers.

Variant

What if the distance function is called multiple times? How to pre-process?

Solution

We can use a hash map to keep track of all positions of a word. When a query comes, it is just to answer the min gap between any num from two sorted arrays.

This can be done by “merging” two sorted array, say a and b: move the smaller pointer and update the min gap if needed.

Another optimization could be when a.length >> b.length, we can scan b and use binary search to find cloest num in a. Complexity is O(mlogn) instead of O(m+n)
