Search Longest String

Problem

Given a dict of strings, D, and a string s. Find the longest string in D that only contains latters in s.

Solution

We just need to encode the strings in D with a 26 bits maps. Then we sort the strings according to length, and from longest to shortest we examine if the bit map is only contains the bit map of s.
