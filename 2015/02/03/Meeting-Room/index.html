Meeting Room

Problem

Given a array of pairs where each pair contains the start and end time of a meeting (as in int), determine if a single person can attend all the meetings

For example: Input array { pair(1,4), pair(4, 5), pair(3,4), pair(2,3) }
Output: false

Solution

The idea is the Leetcode interval problems, we can sort the intervals according to the starting time and scan them to see if two consecutive intervals overlaps.

Variant

Follow up: determine the minimum number of meeting rooms needed to hold all the meetings.
For example, input array { pair(1, 4), pair(2,3), pair(3,4), pair(4,5) }, Output: 2
