Sampling Stream

Problem

How to sample a stream of integer randomly?

Solution

The more fundamental question is how to sample something without knowing the size of data being sampled. The answer is reservoir sampling: we keep track of a candidate; for the k-th element, we keep it with a probability of $\frac{1}{k}$ and keep the candidate with a probability of $\frac{k-1}{k}$.

Variant

Given a binary tree, return a node randomly. Assume the tree is very large.

Solution

The same as stream/list or whatever “very large” data structure. We can always use reservoir sampling without having to scan all the data.
