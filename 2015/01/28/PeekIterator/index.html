PeekIterator

Problem

Implement peek() and pop() from java iterator(). for example [1,2,3,4,5], peek() = 1, pop() = 1, peek() = 2, peek() = 2, pop() = 2

Solution

The idea is to use a normal iterator + an element to store the peek element.
